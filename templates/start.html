Encryp.us is an open-source web chat application built with privacy as a priority. *

  • Encryp.us uses several different encryption techniques to ensure that your communication cannot be read.
  • For group chats, all messages are encrypted with and AES cipher using a session stored password only known to your browser, and never known by the server or those spying on your connection.
  • For one to one chat, all messages are encrypted using an AES cipher and a session key that is automatically generated based on the Diffie-Hellman exchange. Thus, there is no need to exchange your keys via a insecure third party.
  • All passwords are salted and hashed on the server.
  • Encryp.us protects users by using SSL / TLS to ensure encrypted communication between server and client.
